# Java - Basit Notlar & GUI Projesi

Bu repo, java konu notları ve GUI uygulamaları içerir. Amaç hızlıca çalıştırılabilecek, öğrenmeye yönelik örnekler sunmaktır.

## Gereksinimler
- Java JDK 11 veya üstü
- İsteğe bağlı: IDE (IntelliJ, Eclipse) veya Maven/Gradle

## Hızlı Başlangıç
1. Klonlayın:
   git clone https://github.com/Gothel8/Java.git

2. Projeyi IDE ile açın veya terminalde derleyip çalıştırın:
   javac ... && java ...

(Projedeki dosya yapısına göre derleme/çalıştırma komutlarını ayarlayın.)
